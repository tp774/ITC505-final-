Process and Creative Choices

The goal was to create an engaging and functional "Lights Out" game using HTML, CSS, and JavaScript. The grid is dynamically generated to avoid hardcoding, ensuring scalability. For the toggle logic, CSS classes were used to simplify the code and enhance readability.

Challenges and Insights

A significant challenge was ensuring that the game state reset properly during restarts while retaining interactivity. The issue was resolved by reinitializing event listeners and dynamically recreating the grid. This highlighted the importance of handling DOM elements dynamically and avoiding stale references.

Future Enhancements

- Adding a move counter to track player performance.
- Introducing difficulty levels by varying grid sizes.
- Enhancing accessibility with keyboard navigation and screen reader support.
- Including a timer to make the game more competitive.
